## Projets Choisis et Structure du Dépôt

Dans le cadre de ce travail, nous avons choisi d'étudier et de modéliser les trois sujets suivants en Recherche Opérationnelle :

1. **Projet 1 : Vehicle Routing Problem (VRP)**
   - **Objectif** : Minimiser les coûts de transport pour une flotte de véhicules.
   - **Code** : `Projet_1_VRP/code/main.py`

2. **Projet 2 : Ordonnancement de Machines**
   - **Objectif** : Allouer des tâches à des machines pour minimiser le "makespan" (Job Shop Scheduling).
   - **Code** : `Projet_2_Ordonnancement/code/main.py`

3. **Projet 3 : Transport de Poisson (Nouadhibou - Nouakchott)**
   - **Objectif** : Optimisation logistique (VRPTW) pour acheminer le poisson en préservant la chaîne du froid sur 470 km.
   - **Code** : `Projet_3_Transport_Poisson/code/main.py`

## Documents et Présentations

- **`Rapport_Projets_RO.docx`** : Le rapport détaillé contenant l'explication, la modélisation mathématique et les contraintes pour les trois projets.
- **`Presentation_Projets_RO_V2.pptx`** : La présentation PowerPoint officielle du projet, avec illustrations, outils utilisés et étapes de réalisation.

## Prérequis et Exécution

Pour exécuter les codes Python, veuillez installer les dépendances listées dans le fichier `requirements.txt` :

```bash
pip install -r requirements.txt
```

Puis, lancez les scripts Python avec les commandes suivantes :
```bash
python Projet_1_VRP/code/main.py
python Projet_2_Ordonnancement/code/main.py
python Projet_3_Transport_Poisson/code/main.py
```
